# 📼 chronicle

My personal media vault — tracking films, series and books I've watched, 
am watching, or want to watch.

Built on [obsidian-media-starter-vault](https://github.com/resMagi/obsidian-media-starter-vault) — check it out if you want your own.

---

## Setup

1. Clone this repo
2. Open the `chronicle/` folder as a vault in Obsidian
3. Install community plugins
4. Add your OMDb API key in **Settings → Media DB**
